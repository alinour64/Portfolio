File Search

I designed and implemented another application, the file manager. This application runs a binary search on every folder in a directory and displays how many occurrences a certain file exists as well as the location of the files. This project tested my optimization skills because in the first run it would take a long time to find certain files. This was because the program had a time complexity of O(n^2). After implementing a binary search tree method to look for files, the time complexity was reduced to O(log n) which was significantly faster.
